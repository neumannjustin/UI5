Constructor Detail
new sap.m.FacetFilter(sId?, mSettings?)
Constructor for a new FacetFilter.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • showPersonalization : boolean (default: false)
    • type : sap.m.FacetFilterType (default: Simple)
    • liveSearch : boolean (default: true)
    • showSummaryBar : boolean (default: false)
    • showReset : boolean (default: true)
    • showPopoverOKButton : boolean (default: false)
  • Aggregations
    • lists : sap.m.FacetFilterList[] (default)
  • Events
    • reset : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
{string}sId? id for the new control, generated automatically if no id is given
{object}mSettings? initial settings for the new control

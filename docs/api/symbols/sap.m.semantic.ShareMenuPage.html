Class sap.m.semantic.ShareMenuPage

extends SemanticPage
known direct subclasses: DetailPage, FullscreenPage

A ShareMenuPage is a sap.m.semantic.SemanticPage with support for "share" menu in the footer.

new sap.m.semantic.ShareMenuPage(sId?, mSettings?)
Constructor for a new ShareMenuPage

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Aggregations
    • customShareMenuContent : sap.m.Button[]

In addition, all settings applicable to the base type sap.m.semantic.SemanticPage can be used as well.

Parameters:
{string}sId? id for the new control, generated automatically if no id is given
{object}mSettings? initial settings for the new control

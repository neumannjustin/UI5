Namespace sap.ui.Device.system

Provides a basic categorization of the used device based on various indicators.

These indicators are for example the support of touch events, the screen size, the used operation system or the user agent of the browser.

Note: Depending on the capabilities of the device it is also possible that multiple flags are set to true.

Field Summary
desktopIf this flag is set to true, the device is recognized as a desktop system.
phoneIf this flag is set to true, the device is recognized as a phone.
tabletIf this flag is set to true, the device is recognized as a tablet.
Field Detail
desktop: boolean
If this flag is set to true, the device is recognized as a desktop system.

Furthermore, a CSS class sap-desktop is added to the document root element.

phone: boolean
If this flag is set to true, the device is recognized as a phone.

Furthermore, a CSS class sap-phone is added to the document root element.

tablet: boolean
If this flag is set to true, the device is recognized as a tablet.

Furthermore, a CSS class sap-tablet is added to the document root element.

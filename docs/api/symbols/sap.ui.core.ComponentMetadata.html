getManifestEntry(sKey, bMerged?): Object
Returns the manifest configuration entry with the specified key (Must be a JSON object). If no key is specified, the return value is null.

Example: sap.ui.core.Component.extend("sample.Component", { metadata: { manifest: { "my.custom.config" : { "property1" : true, "property2" : "Something else" } } } });

The configuration above can be accessed via sample.Component.getMetadata().getManifestEntry("my.custom.config").

Parameters:
{string}sKey key of the custom configuration (must be prefixed with a namespace / separated with dots)
{boolean}bMerged? whether the custom configuration should be merged with components parent custom configuration.
Since:
1.27.1
Returns:
{Object} custom Component configuration with the specified key.

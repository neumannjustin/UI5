HTML and XML Templating

SAPUI5 supports HTML templating and XML templating.

The HTML templating concept in enables you to bind texts against properties in the model.

XML templating uses an XML view as a template.


In this section:
    HTML Templating
    To build semantic HTML templates, OpenUI5 uses handlebars. You can either use standard expressions and helpers together for the context of a template instance, or you can use the custom helpers for handlebars provided by OpenUI5, which you can use together with the OpenUI5 models.
    XML Templating
    XML templating lets you use an XML view as a template.

Variant for Bootstrapping from Content Delivery Network

OpenUI5 can either be loaded relatively from a Web server or externally from a Content Delivery Network (CDN).

The CDN version of our libraries can be found at https://openui5.hana.ondemand.com/resources/sap-ui-core.js (OpenUI5) and .

If you want to use the CDN version, you can use the following bootstrap script:
#!html<script id="sap-ui-bootstrap"
    type="text/javascript"
    src="https://openui5.hana.ondemand.com/resources/sap-ui-core.js"
    data-sap-ui-theme="sap_bluecrystal"
    data-sap-ui-libs="sap.m"></script>
Caution

The CDN version is constantly being upgraded to the latest stable version. Therefore, this might have an impact on the stability of your application.

You can refer to a specific version instead by using a versioned URL as in the example below:
#!html<script id="sap-ui-bootstrap"
    type="text/javascript"
    src="https://openui5.hana.ondemand.com/1.28.8/resources/sap-ui-core.js"
    data-sap-ui-theme="sap_bluecrystal"
    data-sap-ui-libs="sap.m"></script>

The first segment of the URL after the host name is used to specify a concrete version.

Check the available versions with respective maintenance status at https://openui5.hana.ondemand.com/versionoverview.html.
Note

Only use the Stable version for productive apps. Nevertheless, if you also want to test the Preview version, you are very welcome to send us your feedback!

Related Information
Versioning of OpenUI5
As of OpenUI5 1.6 there is a close coupling of OpenUI5 core/runtime and OpenUI5 tools: It is required that within a running system both have the same version, that is, have the same major and minor version.

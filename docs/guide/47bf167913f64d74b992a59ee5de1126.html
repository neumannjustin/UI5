Using Runtime Default Values for Configuration

OpenUI5 runtime provides default values for the configuration parameters. These default values can be used for the configuration.

The complete list of configuration parameters is available in the API reference under sap.ui.core.Configuration. An overview of the most important configuration parameters and their default values is available in the Configuration Options topic.

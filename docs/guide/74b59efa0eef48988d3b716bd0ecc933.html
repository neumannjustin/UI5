Browser and Platform Support

This sections describes which browsers and platforms are supported by the various OpenUI5 libraries.

As OpenUI5 is based on CSS3, HTML5, and the new JavaScript API, only browsers with HTML5 capabilities are supported.

Depending on the platform your OpenUI5 applications run on, different browsers in different versions are supported. If you know which platform and which browsers are used by your users, you can decide on which libraries to use for your application.

Caution

Microsoft Internet Explorer 11 (IE11) offers an Enterprise Mode that is designed to emulate either Internet Explorer 7 (IE7) or Internet Explorer 8 (IE8). All libraries only support IE11 in Enterprise Mode if they support IE7 and IE8 accordingly.


In this section:
    sap.ui.core, sap.ui.layout, sap.ui.unified
    Basic libraries, supporting all platforms or browsers that are supported by any of the other libraries
    sap.ui.commons, sap.ui.ux3, sap.ui.table
    Browser support for sap.ui.commons, sap.ui.ux3, sap.ui.table libraries on Microsoft Windows and Mac OS platforms.
    sap.m
    Browser support for sap.m library on iOS, Android, BlackBerry, MacOS, and Windows platforms

Linking your Eclipse Editor to the Demo Kit

You can use Quick Fixes to display the API documentation of a OpenUI5 control in the Demo Kit.

Procedure

  1. Place the cursor on the actual OpenUI5 control name in your JavaScript code or in your XMLView. The name of the control in JavaScript code usually starts with sap.
  2. To see all available Quick Fixes, press CTRL+1.
  3. To open the API documentation of the control in the Demo Kit, choose Display in Demo Kit.

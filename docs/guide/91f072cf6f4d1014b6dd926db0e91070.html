sap.ui.commons, sap.ui.ux3, sap.ui.table

Browser support for sap.ui.commons, sap.ui.ux3, sap.ui.table libraries on Microsoft Windows and Mac OS platforms.

Microsoft Windows Platforms
Browser Version Supported
Microsoft Internet Explorer 9 and upwards
Mozilla Firefox Extended Support Release (ESR) and latest version
Google Chrome Latest version
Caution Touch is not supported for Microsoft Windows.
Windows Phone
Browser Version Supported
Internet Explorer None
Opera None
SAP Fiori Client None
Web View None
Mac OS platforms

On Mac OS platforms only Mac OS X is supported with Safari browser version 5.1 and upwards.

Caution Touch is not supported for Mac OS platforms.

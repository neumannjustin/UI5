Dynamic Loading of Libraries

OpenUI5 provides the sap.ui.getCore().loadLibary() method to load libraries at runtime in addition to the libraries declared in the runtime configuration.

After loading, you can use all controls from these libraries. For these additional libraries, the same restriction apply as for the declared libraries: Accessing the document object model (DOM) is only possible after document.ready. Also, rendering applies for these libraries in the same way as for the declared libraries.
